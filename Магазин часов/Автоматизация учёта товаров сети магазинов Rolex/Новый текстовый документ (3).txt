Приветствую тебя дорогой пользователь!
Программа для автоматизации учёта сети товаров магазина Rolex очень проста в применении 
и не требуют каких-либо особых технических навыков для понимания её работы.
При запуске программы откроется окно авторизации. Логин и пароль работникам выдаёт
администратор, покупатели вводятего сами Для того, чтобы авторизоваться, пользователь
вводит свои данные в поля и нажимает на кнопку «Войти». При нажатии на кнопку 
«Выход» - программа закроется. В зависимости от выбора в combobox, у программы разные режимы работы программы. 
Если пользователь - работник, то он сможет добавлять информацию о товарах, искать и фильтровать информацию,
а также удалять заказы. 
При прохождении авторизации, работник перейдёт на главное окно программы. Главное окно 
содержит таблицу с данными из бд, 8 кнопок и 1 combobox. При нажатии на кнопку «Добавить заказ», 
пользователь перейдёт в окно добавления поставки где сможет ввести его наименование, 
выбрать магазин, и ввести цену. При нажатии на кнопку «Сохранить» - данные 
сохранятся в бд, при нажатии на кнопку «Отмена» - окно скроется.
При нажатии на кнопку «Изменить заказ», пользователь перейдёт в окно 
изменения товарв где сможет изменить её наименование, выбрать новую категорию, нового поставщика, 
новое количество и новую цену. При нажатии на кнопку «Изменить» - данные изменятся в бд, при нажатии
на кнопку сохранятся в бд, при нажатии на кнопку «Отмена» - окно скроется. При нажатии на кнопку 
«Удалить поставку», пользователь перейдёт в окно удаления поставки. При выборе поставки и при нажатии 
на кнопку удалятся, данные удалятся из бд, при нажатии на кнопку «Отмена» - окно скроется.
При нажатии на кнопку «Добавить заказ», пользователь перейдёт в окно 
добавления заказа где сможет выбрать заказчика, выбрать категорию, выбрать продукцию и ввести 
цену. При нажатии на кнопку сохранить, данные сохранятся в бд, при нажатии на кнопку «Отмена» - окно скроется.
При нажатии на кнопку «Изменить заказ», пользователь перейдёт в окно 
изменения поставки где при выборе заказа, сможет изменить поставщика, выбрать новую категорию и
ввести новую цену. При нажатии на кнопку «Изменить» - данные изменятся в бд, при нажатии
на кнопку отмена, окно скроется. При нажатии на кнопку «Удаление заказа», пользователь перейдёт 
в окно удаления поставки. При выборе поставки и при нажатии на кнопку удалятся, данные удалятся 
из бд, при нажатии на кнопку «Отмена» - окно скроется. При нажатии на кнопку «отчёт по состоянию склада», 
пользователь перейдёт в окно, где сформируется отчёт. Этот отчёт можно сохранить во многие другие 
форматы. При выборе типа поиска, пользователь может вывести в таблицу информацию, которая его интересуют
при вводе текста, в соответствии с видом поиска и при нажатии на кнопку «поиск». После изменения данных
в бд или после поиска, пользователь может обновить содержимое таблицы после нажатия на кнопку «Обновить».
При нажатии на кнопку «Выход», пользователь выйдет из аккаунта.
Если у пользователя должность заведующий, то он сможет попасть в меню администратора, после нажатии на
кнопку «Режим администратора». В режиме администратора пользователь сможет добавлять, изменять или 
удалять информацию о поставщиках, заказчиках и работниках. Принцип работы точно такой же, как и при 
работе с такими же функциями в главном меню.
